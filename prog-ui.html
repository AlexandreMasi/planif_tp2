Programmation d’interface Web 1

Connaissances

Théoriques (quelles sont les notions que vous avez retenues) ?

Les notions retenues incluent le HTML pour structurer le contenu des pages web, le CSS pour la mise en forme et le design, et le JavaScript pour l'interactivité. Les concepts tels que le modèle de boîte CSS, les sélecteurs et les événements JavaScript sont également importants.

Pratiques (quels sont les logiciels que vous avez appris) ?

Les logiciels appris incluent des éditeurs de code comme Visual Studio Code, des outils de gestion de versions tels que Git et GitHub, et des systèmes de gestion de bases de données comme MySQL. L'utilisation de frameworks comme Vue.js et Laravel a également été développée.

Conditionnelles (quels sont les projets que vous avez réalisés) ?

Les projets réalisés incluent la création d'une application SPA pour afficher les informations météorologiques, le développement d'un site web pour un restaurant pub, ainsi que des projets utilisant des APIs pour récupérer et afficher des données sur les cryptomonnaies et les actions.
