Environnement de développement Web 2

Connaissances

Théoriques (quels sont les notions que vous avez retenues)?

Les concepts fondamentaux des environnements de développement et de production web. Les protocoles de communication courants tels que HTTP, HTTPS, et FTP. Les principes de sécurité des serveurs web, incluant l'utilisation de SSH, de certificats RSA, et de règles de pare-feu. Les notions de virtualisation et la configuration d'hôtes Web virtuels. La gestion des identités et la protection du code source en sécurité.

Pratiques (quels sont les logiciels que vous avez appris)?

Logiciels de développement web comme Visual Studio Code. Outils de virtualisation comme Aws Serveurs web comme Lamp et installation de phpmyadmin node-red mattermost et next cloud ajout de clé SSH et de configuration SFTP Certificats SSL/TLS avec des outils comme Let's Encrypt.

Conditionnelles (quels sont les projets que vous avez réalisés)?

Mise en place d'un environnement de développement local pour un projet web. Déploiement d'une application web sur un serveur distant. Configuration de virtual hosts pour héberger plusieurs sites sur un seul serveur. Sécurisation d'un serveur web en ligne en configurant des pare-feu et en utilisant des clés SSH. Implémentation de certificats SSL pour sécuriser la communication entre le client et le serveur.
