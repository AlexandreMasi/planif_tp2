Initiation a la création de sites web

Connaissances

Théoriques (quels sont les notions que vous avez retenues)?

Les principes de base du langage de balisage HTML pour structurer le contenu des pages Web. Les concepts fondamentaux du CSS pour le style et la présentation des pages Web statiques. Les bases de l'ergonomie et du design d'interfaces Web pour créer des sites intuitifs et faciles à utiliser. Les règles d’utilisabilité et d’accessibilité pour concevoir des sites accessibles à tous les utilisateurs. Les normes de validation en vigueur dans l'industrie pour assurer la qualité et la compatibilité des sites Web.

Pratiques (quels sont les logiciels que vous avez appris)?

Utilisation d'éditeurs de texte comme Visual Studio Code pour écrire et organiser le code HTML et CSS. Exploration de navigateurs Web tels que Chrome et Firefox pour tester et valider le rendu des pages Web. Apprentissage de conformité du code HTML et CSS. Découverte de CSS de base pour styliser rapidement des éléments d'interface.

Conditionnelles (quels sont les projets que vous avez réalisés)?

Création de pages Webs statiques respectant les normes HTML et CSS pour présenter du contenu textuel et visuel. Développement d'un mini-site Web respectant les principes d'ergonomie et de design pour une navigation intuitive. Intégration de médias tels que des images dans une page Web en utilisant les balises HTML appropriées. Application des concepts d’accessibilité en ajustant la structure et le style des pages pour les rendre accessibles aux utilisateurs. Validation et optimisation d'un site Web pour qu'il respecte les normes d'utilisabilité et de conformité.
