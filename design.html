Création et design de sites Web

Connaissances

Théoriques (quelles sont les notions que vous avez retenues) ?

Les notions principales retenues incluent les fondamentaux du HTML pour structurer les pages web, le CSS pour le design et la mise en page, ainsi que les principes de base du JavaScript pour rendre les sites interactifs. J'ai aussi approfondi mes connaissances sur les bonnes pratiques en matière d'accessibilité et de responsive design.

Pratiques (quels sont les logiciels que vous avez appris) ?

J'ai appris à utiliser divers logiciels et outils tels que Visual Studio Code pour l'édition de code, Git et GitHub pour la gestion des versions, et des outils de prétraitement CSS comme SCSS. J'ai également utilisé des frameworks comme Bootstrap pour accélérer le développement et assurer une bonne mise en forme.

Conditionnelles (quels sont les projets que vous avez réalisés) ?

Parmi les projets réalisés, on trouve la création d'un site web pour un restaurant pub, le développement d'une application SPA pour afficher les informations météorologiques, et la réalisation de diverses pages web utilisant des API pour récupérer et afficher des données dynamiques sur les cryptomonnaies et les actions.
