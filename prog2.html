Programmation Web dynamique

Connaissances

Théoriques (quels sont les notions que vous avez retenues)?

Les caractéristiques essentielles d’une base de données et leur importance dans la gestion des données sur un site Web. Les concepts fondamentaux d’un système de gestion de bases de données (SGBD) et son rôle dans la création et la gestion de bases de données. Les méthodes d'analyse des besoins pour créer une base de données adaptée à une application Web dynamique. Les techniques d'optimisation des bases de données pour améliorer les performances et l'efficacité des requêtes.

Pratiques (quels sont les logiciels que vous avez appris)?

Utilisation de MySQL et SQL pour la création, la gestion et l'optimisation des bases de données. Apprentissage de PHP pour la programmation côté serveur et l'interaction avec les bases de données. Utilisation d'outils comme phpMyAdmin pour gérer les bases de données de manière graphique. Développement d'applications Web dynamiques en utilisant un environnement de développement local comme XAMPP ou WAMP.

Conditionnelles (quels sont les projets que vous avez réalisés)?

Création d'une base de données relationnelle pour gérer le contenu dynamique d'un site Web. Développement d'une application Web intégrant des fonctionnalités de gestion de contenu à partir d'une base de données. Rédaction et optimisation de requêtes SQL complexes pour interagir avec une base de données dans un contexte de production. Résolution d'erreurs de requêtes SQL et débogage d'applications Web pour assurer leur bon fonctionnement.
